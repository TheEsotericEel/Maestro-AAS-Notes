# Week 7: Algorithmic Planning and Pseudocode

## 🧠 Core Objectives
This week marks the transition from *learning syntax* to *engineering solutions*. You will move beyond writing code that "just works" to designing algorithms that are robust, logical, and verifiable *before* a single line of Python is written.

### Key Concepts
1.  **Algorithmic Design**: The separation of logic (the plan) from syntax (the code).
2.  **Pseudocode Strategy**: Writing language-agnostic blueprints to solve complex problems.
3.  **Control Flow Logic**: Mastering strict conditional hierarchies (If/Else/Elif).
4.  **Verification**: Using trace tables (dry runs) to mathematically prove your logic is correct.
