## 📖 The Glossary

### Data Types (The Nouns)
| Type | Name | Example | Properties |
| :--- | :--- | :--- | :--- |
| `str` | String | `"Hello"` | Text. Immutable. |
| `int` | Integer | `42` | Whole numbers. Precision. |
| `float` | Floating Point | `3.14` | Decimals. Approximate. |
| `bool` | Boolean | `True` | Logic state (On/Off). |

### Operators (The Verbs)
| Symbol | Name | Description |
| :--- | :--- | :--- |
| `=` | Assignment | Stores value in variable. (Right to Left). |
| `+` | Addition | Adds numbers OR Concatenates strings. |
| `/` | True Division | Returns `float`. |
| `//` | Floor Division | Returns `int` (Quotient). |
| `%` | Modulo | Returns Remainder (Leftover). |
| `**` | Exponent | Power ($x^y$). |

### Functions (The Actions)
| Function | Description | Return Type |
| :--- | :--- | :--- |
| `print(...)` | Displays text to console. | `None` |
| `input(...)` | Pauses program to get type text. | `str` |
| `type(...)` | Checks the class of a value. | `type` |
| `int(...)` | Casts to Integer (Truncates). | `int` |
| `round(...)` | Rounds to nearest Even. | `int` |
